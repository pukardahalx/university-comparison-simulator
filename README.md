# Engineering Choice Simulator

An interactive decision matrix designed to evaluate undergraduate engineering programs at major Italian universities.

## Key Features
* **Dynamic Ranking Algorithm:** Real-time re-ordering of universities based on user-defined priority weights.
* **Custom Metrics:** Evaluates programs based on Hardware/Physical Layer, Software/Algorithms, Telecommunications, Math/Physics Rigor, and Academic Ecosystem.
* **Data-Driven:** Scoring is derived from heuristic calculations of ECTS/CFU allocations between Italian L-8 and L-31 macro-classes.

## Tech Stack
* **Frontend:** HTML5, CSS3 (Grid/Flexbox), and Vanilla JavaScript.
* **Architecture:** Single-file architecture requiring no external dependencies or build steps.

## Usage
This project is a static web application. It is optimized for deployment via GitHub Pages or local browser execution.
